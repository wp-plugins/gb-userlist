== Description ==

Display list of your wordpress users into your sidebar.
Display as list or using avatar's user.
You could only display recent users or select users randomly.

* French translation include.

== Installation ==

1. Just put the plug-in into your plug-in directory and activate it.
2. Add new widget on your sidebar
